# 部署方案选择指南

## 快速决策

| 你的情况 | 推荐方案 |
|---------|---------|
| 新手、想快速上线 | **Render** |
| 国内用户为主 | **阿里云轻量服务器** |
| 国际用户、需要 CI/CD | **Railway** |
| 有服务器经验、追求性价比 | **Docker + VPS** |

---

## 方案详解

### 1. Render（推荐新手）

**适合场景：**
- 第一次部署项目
- 不想处理服务器配置
- 需要免费试用

**优点：**
- ✅ 零配置，自动从 GitHub 部署
- ✅ 免费 SSL 证书
- ✅ 自动重启
- ✅ 简单的 Web 界面

**缺点：**
- ❌ 免费版 15 分钟无活动会休眠
- ❌ 国内访问较慢

**成本：**
- 免费: 休眠制
- $7/月: 持续运行

**预计时间：** 10-15 分钟完成部署

**详细教程：** [render-deploy.md](render-deploy.md)

---

### 2. 阿里云轻量服务器（推荐国内用户）

**适合场景：**
- 主要面向国内用户
- 需要稳定的访问速度
- 长期运营

**优点：**
- ✅ 国内访问速度快
- ✅ 完全控制服务器
- ✅ 价格实惠
- ✅ 可以部署多个项目

**缺点：**
- ❌ 需要手动配置
- ❌ 需要基本的服务器运维知识

**成本：**
- ¥24/月 (1核1GB)
- 约 ¥300/年

**预计时间：** 30-60 分钟完成部署

**详细教程：** [aliyun-deploy.md](aliyun-deploy.md)

---

### 3. Railway

**适合场景：**
- 需要 CI/CD
- 需要数据库支持
- 团队协作

**优点：**
- ✅ 自动部署
- ✅ 内置 PostgreSQL
- ✅ 支持多种语言
- ✅ 优秀的开发者体验

**缺点：**
- ❌ 价格稍贵
- ❌ 国内访问较慢

**成本：**
- 免费试用 $5 额度
- $5/月起

**预计时间：** 10-15 分钟完成部署

---

### 4. Docker + VPS

**适合场景：**
- 有服务器运维经验
- 需要完全控制
- 追求性价比

**优点：**
- ✅ 环境一致性
- ✅ 易于迁移
- ✅ 资源利用率高

**缺点：**
- ❌ 需要学习 Docker
- ❌ 配置相对复杂

**成本：**
- VPS: $3-5/月

**预计时间：** 20-30 分钟完成部署

**详细教程：** [docker-deploy.md](docker-deploy.md)

---

## 部署前检查清单

无论选择哪种方案，部署前请确认：

### 代码准备
- [ ] 代码已推送到 GitHub
- [ ] .env 文件不在仓库中
- [ ] .gitignore 配置正确
- [ ] 前端 API 地址可配置

### 环境变量准备
- [ ] LLM_API_KEY 已获取
- [ ] JWT_SECRET_KEY 已生成
- [ ] 数据库方案已确定

### 安全准备
- [ ] 已更换默认密钥
- [ ] 已配置 HTTPS（生产环境）
- [ ] 已设置防火墙规则

---

## 部署后必做事项

1. **测试所有功能**
   - 用户注册/登录
   - 核心功能流程
   - API 响应速度

2. **配置监控**
   - 设置错误报警
   - 配置日志收集
   - 监控服务器资源

3. **备份数据**
   - 数据库定期备份
   - 配置自动备份

4. **性能优化**
   - 启用 CDN
   - 配置缓存
   - 压缩资源

---

## 我的推荐

**如果你是第一次部署：**
1. 先用 **Render** 免费版练手

**如果要正式运营：**
1. 直接使用 **阿里云轻量服务器**
2. 配置域名和 SSL
3. 设置自动备份
